## Application Features

- Convert Word (.docx) to Elementor JSON
- Intelligent heading detection (H1-H6)
- Image extraction with positioning
- Table detection and HTML conversion
- Multi-column layouts (1, 2, or 3)
- Multiple distribution strategies
- Professional UI without decorative icons
- Integrated user guide
- Complete credits system

---

## Workflow Summary

1. **Prepare WordPress**: Get Media Library URL
2. **Configure**: Set URL, layout, and distribution
3. **Upload**: Select Word document
4. **Convert**: Click convert button
5. **Download**: Get JSON and images
6. **WordPress**: Upload images and import template
7. **Publish**: Use template in Elementor

Detailed instructions in [USER_GUIDE.md](USER_GUIDE.md)
